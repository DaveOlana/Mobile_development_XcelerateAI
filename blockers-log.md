## 🔍 How to Log a Blocker
When encountering a blocker, document it using the following format:
1. **Symptom:** What error or behavior did you observe?
2. **Analysis:** What do you think is causing it?
3. **Attempts:** What did you try?
4. **Resolution:** How was it solved?
